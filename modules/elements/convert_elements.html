static from_kozai(e, i, n)[source]#

Converts from Kozai mean motion to Brouwer mean motion

Parameters:

  • e (float) – The eccentricity of the orbit

  • i (float) – The inclination of the orbit in \(deg\)

  • n (float) – The Kozai mean motion of the orbit in \(\frac{rev}{day}\)

Return type:

float

Returns:

The Brouwer mean motion of the orbit in \(\frac{rev}{day}\)

Example:

from pysaal.elements import ConvertElements

brouwer_n = ConvertElements.mean_motion.brouwer.from_kozai(0.0001, 0.42, 42)

EquinoctialElements

cart = CartesianElements(

    -95.4631435846219,
    0.10749110300482166,
    -3.072941987275631,
    0.021437245884576635,
)
eqnx = ConvertElements.equinoctial.from_cartesian(cart, Earth.get_mu())

static from_sp_vector(sp, tle_type)[source]#

Converts from a SPVector to a TLE

Parameters:

  • sp (SPVector) – The SPVector to convert

  • tle_type (TLEType) – The type of TLE to generate

Return type:

TLE

Returns:

The TLE of the satellite

Example:

from pysaal.elements import ConvertElements, SPVector, CartesianElements
from pysaal.time import Epoch
from pysaal.enums import TLEType

epoch = Epoch.now()
teme_state = CartesianElements(
    -42134.866555365596,
    -1477.3611216130757,
    -95.4631435846219,
    0.10749110300482166,
    -3.072941987275631,
    0.021437245884576635,
)
sat_id = 1
sp = SPVector(epoch, teme_state, sat_id)
tle_type = TLEType.SGP4
tle = ConvertElements.tle.from_sp_vector(sp, tle_type)
